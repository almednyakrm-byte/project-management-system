# نظام إدارة المشاريع (تطوير، ادارة الوقت، تحليل)
==========================

## Overview & Project Purpose
---------------------------

نظام إدارة المشاريع هو تطبيق مفتوح المصدر يهدف إلى تسهيل إدارة المشاريع من خلال توفير أدوات متكاملة لتحليل الوقت، تطوير المشروع، وتحليل الأداء. يهدف هذا المشروع إلى توفير بيئة عمل متكاملة ومريحة للمشاريع الصغيرة والكبيرة على حد سواء.

### Module Overview

*   **تطوير المشروع**: يتيح هذا الموديل إدارة المشاريع من خلال توفير أدوات لإنشاء مشاريع جديدة، إضافة أعضاء للمشروع، وتحديث حالة المشروع.
*   **إدارة الوقت**: يتيح هذا الموديل إدارة الوقت للمشاريع من خلال توفير أدوات لإنشاء سجلات الوقت، إضافة سجلات جديدة، وتحديث حالة السجلات.
*   **تحليل الأداء**: يتيح هذا الموديل تحليل الأداء للمشاريع من خلال توفير أدوات لإنشاء تقارير الأداء، إضافة تقارير جديدة، وتحديث حالة التقرير.

### Tables Overview

*   **projects**: يحتوي على جميع المشاريع الموجودة في النظام.
*   **project_members**: يحتوي على جميع أعضاء المشاريع الموجودة في النظام.
*   **time_entries**: يحتوي على جميع سجلات الوقت الموجودة في النظام.
*   **performance_reports**: يحتوي على جميع تقارير الأداء الموجودة في النظام.

### Roles Overview

*   **admin**: يمتلك جميع الصلاحيات في النظام.
*   **project_manager**: يمتلك صلاحيات إدارة المشاريع.
*   **team_member**: يمتلك صلاحيات إضافة سجلات الوقت.

## Step-by-Step Instructions
---------------------------

### Running the Environment using Docker-Compose

1.  قم بتحميل المشروع على جهازك.
2.  قم بتشغيل الأمر التالي في терمينال:

    bash
docker-compose up


3.  قم بتشغيل الأمر التالي في ترمينال لتشغيل المكونات:

    bash
docker-compose up -d


4.  قم بتشغيل الأمر التالي في ترمينال لتشغيل المكونات في الخلفية:

    bash
docker-compose up -d --build


5.  قم بتشغيل الأمر التالي في ترمينال لتشغيل المكونات في الخلفية مع بناء الصور:

    bash
docker-compose up -d --build --no-deps
